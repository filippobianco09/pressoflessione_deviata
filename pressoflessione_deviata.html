Tipo di sezione
Materiali (resistenze di calcolo)
NTC 2018: λ = 0.8 (fck ≤ 50 MPa), η = 1.0. Blocco rettangolare equivalente.
εcu = 3.5‰, εyd = fyd/Es (Es = 200 000 MPa).
Sollecitazioni di progetto
NEd < 0 = compressione. Mx = momento attorno all'asse x (orizzontale). My = momento attorno all'asse y (verticale).
Armature — Layer
Doppia simmetria. Ogni layer definisce una fascia di armatura simmetrica: le barre sul lato inferiore (quota cinf dal bordo) e sul lato superiore sono identiche, così come quelle sui due lati verticali (quota clat dal bordo). Gli angoli sono condivisi una sola volta. Totale barre = 2·nx + 2·ny − 4.
Distribuzione uniforme a 360°. Ogni layer è una corona di n barre equidistanti posizionate a distanza c dal bordo esterno. Il centro delle barre è a raggio R − c dal centro della sezione.

Metodo di calcolo
Algoritmo iterativo — Asse neutro inclinato

La verifica è condotta per integrazione numerica della sezione discretizzata in strisce orizzontali sottili (Δy = H/500).

Per ogni tentativo di asse neutro (inclinazione α, profondità dn), si calcola:
• Deformazione di ogni striscia di calcestruzzo → tensione con blocco rettangolare equivalente (λ=0.8, η=1.0, NTC 2018 par. 4.1.2.1.2.1)
• Deformazione di ogni layer di acciaio → tensione con modello elastico-perfettamente plastico (±fyd)
• Equilibrio: Ncalc, Mx,calc, My,calc

La convergenza è cercata su (α, dn) con metodo di Newton-Raphson 2D fino a residuo < 0.1%.

Dominio Mx–My: per NEd fissato, si ruota la direzione del momento di 0°→360° e si trova il momento resistente radiale → curva chiusa di capacità.

Coefficiente di sicurezza: rapporto tra il momento resistente nella direzione del momento applicato e il momento applicato totale.

Convenzioni segni: N < 0 = compressione, asse y verso l'alto, asse x orizzontale.
Limiti e avvertenze
• Calcolo allo stato limite ultimo (SLU)
• Non si considerano effetti del secondo ordine (instabilità, imperfezioni geometriche)
• Armatura minima/massima non verificata automaticamente
• Le tensioni di trazione nel calcestruzzo sono trascurate (conservativo)
• Risultati validi per sezioni a comportamento simmetrico nelle armature
• Uso esclusivamente a scopo indicativo — verificare sempre con un professionista abilitato
